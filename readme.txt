Решается задача мультилейбловой классификации с помощью предобученной модели обработки естественного языка (NLP) "DeepPavlov/rubert-base-cased"

predict.ipynb - считывает веса обученной модели и выдает ответ для приватного датасета private_test.csv без обучения модели
train_and_predict_public_Task3_final24_seed_110.ipynb - код для обучения модели и запуска предсказания и формирования .csv файла на тестовом наборе
Task3_final24_Bert_25_split_LB_64.232_20epoch_seed_110_check.pth - веса обученной модели

Проверка гипотез:
Решение задачи с помощью нейронной сети и аугментации датасета с помощью синонимов.

Обучение проводилось на увеличенном за счет разбивки длинных описаний датасете, обработка датасета происходит в train_and_predict_public_Task3_final24_seed_110.ipynb.
